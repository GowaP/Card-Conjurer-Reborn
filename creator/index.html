Comment sont sauvegardées mes cartes ?
Les cartes sont enregistrées sur votre ordinateur dans le stockage local de votre navigateur, qui est généralement limité à 5 Mo et ne peut être modifié.
Malheureusement, cela signifie que si vous enregistrez beaucoup de cartes, vous pourriez manquer d'espace.
Votre stockage local manque d'espace particulièrement vite lorsque vous téléchargez des des images directement depuis votre ordinateur, car l'image elle-même doit être enregistrée. Cependant, si possible, le téléchargement d'images via une URL vous fera gagner énormément d'espace, ce qui vous permettra de sauvegarder beaucoup plus de cartes. de sauvegarder beaucoup plus de cartes.
Et si vous manquez d'espace, ne vous inquiétez pas ! Vous pouvez télécharger toutes les cartes enregistrées puis supprimer toutes les cartes enregistrées, libérant ainsi les 5 Mo d'espace. Et lorsque vous Et lorsque vous souhaitez modifier les cartes que vous avez précédemment téléchargées/supprimées, vous pouvez les télécharger à nouveau via la fonction de téléchargement de fichiers (sous "Télécharger les cartes précédemment enregistrées"). téléchargement de fichiers (sous "Télécharger les cartes précédemment téléchargées").
